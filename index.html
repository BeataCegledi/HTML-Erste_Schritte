Anchor-Tags (<a>) - Links in HTML

Mit den Anchor-Tags können wir Links erstellen. Links führen zu anderen Seiten, Dateien oder zu einer Stelle auf derselben Seite.

1. Grundaufbau eines Links

So sieht ein einfacher Link im Code aus:

                    <a href="https://www.google.com">Ich bin ein Link</a> 
                
                 /\_/\\
                ( o.o )
                 > ^ <

So sieht der Link im Browser aus:

Ich bin ein Link

2. Das Attribut href

href steht für Hypertext Refrence. Das Attribut href gibt an, wohin der Link führt. Ohne href ist es KEIN richtiger Link!

  • href="./010_hallo.html" → Link zu einer Datei im gleichen Ordner
  • href="../010_hallo.html" → Link zu einer Datei in einem übergeordneten Verzeichnis
  • href="../../010_hallo.html" → Link zu einer Datei in einem übergeordneten Verzeichnis
  • href="/010_hallo.html" → Startet im Hauptverzeichnis der Webseite (Root)
  • href="#ziel" → Link zu einer Stelle auf derselben Seite
  • href="https://www.google.com" → Link der zu einer anderen Webseite führt

3. Link in einem neuen Tab öfnen

Mit target="_blank" öffnet sich der Link in einem neuen Tab: 

                    <a href="https://www.google.com" target="_blank" rel="noopener noreferrer>Ich bin ein Link</a>

So sieht ein Link im Browser aus:

Ich bin ein Link

noopener
Sicherheit bei target="_blank", das die Zielseite, meine Seite NICHT manipulieren kann.

noreferrer
Datenschutz, bedeutet die Zielseite sieht NICHT, von welcher Seite der Klick kam.

rel
eng. relationship / dt. Beziehung

4. E-Mail und Telefon Links

E-Mail (mailto)

                    <a href="mailto:beispiel@gmail.com">E-Mail schreiben</a>

Unser Browser öffnet das Standard E-Mail Programm des Nutzers. Und erstellt automatisch: Empfänger (E-Mail ist schon ausgefüllt), Betreff: leer, Text: leer.

E-Mail schreiben

Telefon (tel)

                    <a href="tel:+491234567890">Jetzt anrufen</a>

Smartphones und Tablets öffnen direkt die Telefon App zum Anrufen. Pcs können die Funktion nur nutzen, wenn die dafür zuständige App/ Programm vorhanden ist.

Jetzt anrufen
