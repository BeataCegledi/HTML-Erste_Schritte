Anchor-Tags (<a>) - Links in HTML

Mit dem Anchor-Tags können wir Links erstellen. Diese können entweder zu anderen Webseiten führen, zu anderen Stellen auf der gleichen Seite oder auch zu Dateien wie PDFs oder Bildern.

1. Grundaufbau eines Links

So sieht ein einfacher Link im Code aus:

                
                    <a href="https://google.com"> Ich bin ein Link </a>
                
                /\_/\\
                ( o.o )
                > ^ <

So sieht der Link im Browser aus:

Ich bin ein Link

2. Das href-Attribut

Das href steht für "hypertext reference". Das Attribut href gibt die Zieladresse des Links an. Ohne das href-Attribut funktioniert der Link NICHT.

Es gibt verschiedene Möglichkeiten, die Zieladresse anzugeben:

  • href="./010_hallo.html" Link zu einer Datei im gleichen Ordner /Webseite (relative URL)
  • href="../010_hallo.html" Link zu einer Datei in einem übergeordneten Ordner (relative URL)
  • href="../../010_hallo.html" Link zu einer Datei in einem zweiten übergeordneten Ordner (zweimal nach draußen)
  • href="/010_hallo.html" Link zu einer Datei im Hauptverzeichnis der Webseite (relative URL)
  • href="C:\pfad\zu\datei.html" Link zu einer Datei auf dem lokalen Computer (lokaler Pfad)
  • href="https://google.com" Link zu einer externen Website (absolute URL)
  • href="/seite.html" Link zu einer Seite im Wurzelverzeichnis (interne URL)
  • href="/datei.pdf" Link zu einer Datei (Datei-Link)

3. Link in einem neuen Tab öffnen

Mit dem target-Attribut können wir steuern, in welchem Tab oder Fenster der Link geöffnet wird.

  • target="_self" Link wird im gleichen Tab geöffnet (Standard)
  • target="_blank" Link wird in einem neuen Tab geöffnet
                    <a href="https://google.com" target="_blank" rel="noopener noreferrer"> Ich bin ein Link, der in einem neuen Tab geöffnet wird </a>

So sieht der Link im Browser aus:

Ich bin ein Link, der in einem neuen Tab geöffnet wird

noopener
Sicherheit BEI target="_blank"; das die Zielseite meine Seite nicht manipulieren kann (z.B. durch JavaScript)

noreferrer
Datenschutz, bedeutet die Zielseite sieht nicht, von welcher Seite der Besucher kommt (keine Referrer-Informationen)

rel
eng Beziehung zwischen der aktuellen Seite und der Zielseite;
z.B. "nofollow" für Suchmaschinen, "noopener" für Sicherheit, "noreferrer" für Datenschutz

4. Email und Telefon-Links

Email-Links (mailto:)

Mit dem mailto: können wir einen Link für eine neue E-Mail in dem standard E-Mail-Client öffnen. 

                    <a href="mailto:beispiel@gmail.com"> E-Mail schreiben </a>
E-Mail schreiben

Wir können auch einen Betreff und eine Nachricht vorgeben, die automatisch in der E-Mail ausgefüllt werden, wenn der Link angeklickt wird.
- %0A steht für einen Zeilenumbruch in der Nachricht.
- %20 steht für ein Leerzeichen.
- %2C steht für ein Komma.
- %3A steht für einen Doppelpunkt.
- %40 steht für das @-Zeichen.
- %3F steht für ein Fragezeichen.
- %E2%82%AC steht für ein Euro-Zeichen.

E-Mail schreiben mit Betreff und Nachricht

Telefon-Links (tel:)

Mit dem tel: können wir einen Link erstellen, der ein Telefonat initiiert.
Dieser Link öffnet auf Handys und Tablets direkt die Telefon-App.
Auf PCs nur dann, wenn dafür geeignete Applikationen verfügbar sind. 

                    <a href="tel:+49123456789"> Rufen Sie uns an </a>
Rufen Sie uns an

5. Interne Links (Anker-Links)

Mit internen Links / Anker-Links kann man zu bestimmten Abschnitten innerhalb derselben Seite springen. 

                    <a href="#ziel"> Zum Ziel springen. Dafür braucht man das Ziel-Element (id="ziel") </a>
                    
<section id="ziel"> So wird das Ziel gekennzeichnet </section>
Zum Ziel springen 
                    <a href="./030_struktur_elemente.html#hinweis"> Zum Hinweis in einer anderen Seite springen. 
                    
 Dafür braucht man das Ziel-Element in dem anderen Html-Datei (id="hinweis") </a>
Zum Hinweis in einer anderen Seite springen

Du kannst auch zu einem bestimmten Abschnitt auf einer anderen Seite springen,
indem du die URL der Seite mit einem # und der ID des Ziel-Elements kombinierst.
Das funktioniert sogar direkt in Browsern: https://de.wikipedia.org/wiki/Basilisk_(Mythologie)#Aachen 

<a href="https://de.wikipedia.org/wiki/Basilisk_(Mythologie)#Aachen" target="_blank" rel="noopener noreferrer">Link zu einem Abschnitt auf einer anderen Seite (Wikipedia) </a>
