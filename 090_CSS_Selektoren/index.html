Selektoren in CSS

Selektoren sind "Auswähler", sie bestimmen, welche HTML-Elemente du mit CSS gestalten kannst.

Erklärung der CSS Selektoren

1. Universalselektor: *

Was macht er? Der wählt wirklich alles aus Jedes einzelne Element der Seite wird ausgewählt.

Wann ist er nützlich? vor allem am Anfang, wenn man Grundeinstellungen für die ganze Seite setzen möchte. 

                        
                            *{
                                font-family:Arial, Helvetica, sans-serif;
                                line-height: 1.5;
                                box-sizing: border-box; 
                                padding: 0;
                                margin: 0;              
                            }

2. Elementselektor: body oder article

Was macht er? Er spricht alle HTML Elemente eines bestimmten Typs an.

Wann benutzen wir ihn? Wenn alle gleichen Elemente gleich aussehen sollen. 

                        body{
                            background: linear-gradient(to bottom, #f606da, #f8fcff);
                            color: #1b0677;
                        }

3. Klassenselektor .class

Was macht er? Er wählt alle Elemente aus, die dieselbe Klasse haben.

Warum ist das praktisch? Weil man damit mehrere Elemente gemeinsam
gestalten kann, auch wenn diee unterschiedliche Tags haben. 

                        .wichtig{
                            color: deeppink;
                        }

4. ID-Selektor #id

Was macht er? Wählt genau ein Element aus mit einer Bestimmten ID.

WICHTIG: Eine ID sollte nur EINMAL auf einer Seite vorkommen. 

                          
                            #wichtig{
                                color:#0cf52f
                            }

5. Gruppenselektor: label, h2, input

Was macht er? Fasst mehrere Seektoren zusammen

Warum ist das hilfsreich?Weil du denselben CSS-code nicht doppelt schreiben musst. 

                            h2,h3{
                                color: #f7a71d;
                            }

6. Nachfahrenselektor: article p

Was macht er? Wählt alle p Elemente, die sich innerhalb eines article befinden

WICHTIG: Es ist egal, ob das p direkt drin ist oder noch tiefer verschächtelt. 

                        
                            article strong{
                                color: #e083e5;
                            }

7. Kindselektor: article > h3

Was macht er? Er wählt nut die h3 Elemente aus,die DIREKT in einer article liegen.

WICHTIG Nur DIREKTE Kinder werden ausgewählt, keine Enkel und keine Urenkel! 

                        
                            article > h3{
                                color:darkgreen;
                                text-decoration: underline wavy;
                            }

8. Direkte Nachbarselektor: h1 + p

Was macht er? Er wählt den ERSTEN p Element direkt nach einem h1 aus.

WICHTIG Wirklich nur der DIREKTE Nachbar wird ausgewählt! 

                        
                            h1 + p{
                                color: seashell;
                            }

9. Allgemeiner Geschwisterselektor: h3 ~ p

Was macht er? Er wählt alle p Elemente aus, die nach einem h3> kommen und auf der selben Ebene liegen.

Unterschied zu + : Hier wird nicht nur der direkte Nachbar gewählt, sondern mehrere passende Geschwister. 

                        
                            h3 ~ p{
                                color:darkslateblue;
                            }

10. Attributselektor: input[type="password"]

Was macht er? Er wählt nur Elemente aus, die ein bestimmtes Attribut besitzen.

Wofür ist es gut? Besonders praktisch für Formulare. 

                        
                            input[type="password"]{
                                border: 2px solid tomato;
                            }

Übersicht: CSS Selektoren

Selektor Beispiel Beschreibung
Universalselektor * Wählt alle Elemente aus
Elementselektor p Wählt alle <p>-Tags aus
Klassenselektor .class Wählt alle Elemente aus mit der Klasse .class
ID-Selektor #id Wählt alle Elemente mit der id="id" aus
Gruppenselektor label, input Wählt alle label und alle input aus
Nachfahreselektor article p Wählt alle <p>Elemente, die sich irgendwo innerhalb eines <article> befinden.
Kindselektor section > p Wählt nur <p>, die direkt im <section> liegen
Direkte Nachbarselektor h1 + p Wählt das ERSTE <p> direkt nach einem <h1>
Allgemeine Geschwisterselektor h1 ~ p Wählt alle <p>Elemente, die nach einem <h1> auf derselben Ebene kommen.
Attributselektor input[type="password"] Hier wird nur das <input> mit dem type="password" angesprochen.
