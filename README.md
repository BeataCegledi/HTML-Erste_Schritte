# 🌐 HTML – Erste Schritte

> Eine kleine Sammlung kommentierter HTML-Lernbeispiele – entstanden im Rahmen der Umschulung zur Fachinformatikerin Anwendungsentwicklung.

## 📋 Projektbeschreibung

Dieses Repository dokumentiert meine ersten Schritte mit HTML und CSS.
Jede Datei behandelt ein Thema mit ausführlichen Kommentaren direkt im Code,
sodass die Beispiele auch als Nachschlagewerk dienen können.

Die `index.html` dient als Übersicht und verlinkt auf die einzelnen Themenseiten.
Von jeder Themenseite führt ein **„← Startseite"**-Link zurück zur Übersicht.

## 📂 Inhalt

| Datei | Thema | Was wird gezeigt? |
|---|---|---|
| `index.html` | Startseite | Übersicht und Navigation zu allen Themen |
| `020_boilerplate.html` | Grundgerüst | `<!DOCTYPE>`, `<html>`, `<head>`, `<body>`, Meta-Tags, Viewport |
| `030_struktur_elemente.html` | Strukturelemente | `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>` |
| `040_links.html` | Links / Anker-Tags | `<a href>`, relative & absolute URLs, `target`, `mailto:`, `tel:`, Anker (`#id`) |

## 💡 Behandelte HTML-Konzepte

| Konzept | Anwendung im Projekt |
|---|---|
| Semantische Elemente | `<header>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>` |
| Überschriften-Hierarchie | `<h1>` bis `<h6>` mit Erklärung zur Verwendung |
| Textauszeichnung | `<p>`, `<br>`, `<strong>`, `<b>`, `<i>`, `<u>`, `<code>`, `<pre>` |
| Listen | `<ul>` mit `<li>` |
| Links | interne, relative, absolute, `mailto:`, `tel:`, Sprungmarken mit `#id` |
| HTML-Entities | `&lt;`, `&gt;`, `&copy;`, `&amp;` |
| Meta-Angaben | `charset`, `viewport` (Responsive Design) |

## ▶️ Verwenden

Einfach `index.html` im Browser öffnen – z.B. durch Doppelklick auf die Datei
oder per Rechtsklick → *„Öffnen mit"* → Browser.

> Tipp: In VS Code mit der Erweiterung **Live Server** lässt sich die Seite
> bequem mit Auto-Reload anzeigen (Rechtsklick auf `index.html` → *Open with Live Server*).

**Voraussetzungen:** Ein beliebiger moderner Browser – sonst nichts.
